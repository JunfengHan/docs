# 视频剪辑基础

## 1. 视频是什么？

> 有声视频由**运动的画面** + **声音**组成。

## 2. 如何生产出视频

两种途径：

- 动画：通过软件制作出运动的画面 + 配音
- 影像记录：使用摄像机/手机拍摄出运动的画面 + 声音

## 3. 视频基础概念

- 码率
- 编码格式
- 封装格式
- 音频
