## 问答

**为何文章那么长，明明可以分开成多个文章啊？**

> 某个问题理解不透彻，主要原因可能不在于这个问题，而是你根本不知道为什么会有这个问题。就像微积分很难，大家压根不知道牛顿为啥创造出微积分一样，或许知道微积分的来历对大家理解它有很大的帮助。因此，我认为应该把一个知识点从头到尾地梳理一下，问题往往会迎刃而解。
>
> 当然，你可以只看自己感兴趣的部分，那些对你来说是浪费时间的部分请跳过。
